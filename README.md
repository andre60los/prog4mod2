
Una aplicación de línea de comandos para administrar tu biblioteca personal usando SQLite.

Funcionalidades

- Agregar libros con título, autor, género y estado de lectura
- Actualizar información de libros
- Eliminar libros
- Ver el listado completo
- Buscar por título, autor o género

Requisitos

- Python 3.x

Ejecución

```bash
python biblioteca.py
